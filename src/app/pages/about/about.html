🎨 Color Studio

A modern Angular application for creating, converting and exploring beautiful colors.

🚀 Project Overview

Color Studio is a collection of professional color tools built with Angular 21. It helps designers and developers generate palettes, create gradients, convert colors and experiment with beautiful color combinations.

🛠 Included Tools

🎨 Color Picker

Pick colors with live preview and multiple color formats.

🌈 Palette Generator

Create beautiful color palettes instantly.

🌅 Gradient Generator

Design linear, radial and conic gradients.

🔄 Color Converter

Convert between HEX, RGB and HSL formats.

💻 Technologies Used

Angular 21 TypeScript HTML5 CSS3

✨ Features

  • Live Color Preview
  • Glassmorphism UI
  • Responsive Design
  • Smooth Animations
  • Copy to Clipboard
  • Random Generators
