# 🎨 Color Studio - Angular 21

> A modern, responsive, and interactive color toolkit built with **Angular 21**. Color Studio helps designers and developers create, convert, and explore colors through powerful tools including a Color Picker, Palette Generator, Gradient Generator, and Color Converter.

---

## 📖 Overview

Color Studio is a professional Angular application designed to simplify color-related tasks for designers and developers. It offers an intuitive interface with real-time previews, multiple color conversion formats, palette generation, gradient creation, and modern glassmorphism UI.

The project demonstrates modern Angular development practices using **Standalone Components**, **Services**, **TypeScript**, and **Responsive Design**.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/charanepuri/color-studio-angular.git
```

Navigate into the project

```bash
cd color-studio
```

Install dependencies

```bash
npm install
```

Run the application

```bash
ng serve
```

Open your browser

```text
http://localhost:4200
```
